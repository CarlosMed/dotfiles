# WIMILI
WIMILI is a Windows, Mac, and Linux Installation for when i do a
clean install.

## Installing WIMILI
---
```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/CarlosMed/dotfiles/master/installs/setup.sh)"
```
